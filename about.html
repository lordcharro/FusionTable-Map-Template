A little how to:


Map Info

It is possible to interact with the map and click on the several sites and obtain the detailed information.

  • The new sites are red with a N
  • The upgrades are blue with a U

We can see all the necessary information for each site.

Search

We can filter the data by:

  • Address
  • Operator
  • Site ID
  • Canton
  • Type of project

Map

When searching by address it’s possible to add a radius of ½ Km, 1Km, 2Km, 5Km, 10Km and 100Km.

It’s also possible to combine all the search fields, ex: The sites from Swiss that are NewSite and only in the Vaud Canton.
